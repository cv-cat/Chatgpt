# Chatgpt
快速部署基于 chatgpt 的网页聊天机器人，下载即用！
基于 chatgpt 的`gpt-3.5-turbo`的 api 的一个对话机器人

2. api已被墙，使用和部署请使用魔法
3. 历史记录功能已删除
